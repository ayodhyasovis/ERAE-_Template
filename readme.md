# Unofficial LaTeX Template for ERAE

This is an unofficial LaTeX template designed for preparing manuscripts for submission to the **European Review of Agricultural Economics (ERAE)**. 

## Disclaimer
This template is **not** an official publication of Oxford Academic or the ERAE. It was created in accordance with the "Information for Authors" guidelines to assist researchers in formatting their work. Always check the [Official ERAE Author Guidelines (https://academic.oup.com/erae/pages/General_Instructions) for the most current requirements.

## License & Usage
- **Edit Freely**: This template is available for free use, modification, and distribution.
- **Accuracy**: While designed for accuracy, the author is not responsible for desk rejections due to formatting.

## Key Submission Requirements
Based on journal guidelines, users of this template must remember:
* **Length**: Articles should not exceed 8,000 words, including abstract, tables, and figures[cite: 9].
* **Anonymity**: Author names and affiliations must appear **only on the first page** to protect the anonymous review process. 
* **Abstract**: Must be no more than 100 words and include conclusions and methods.
* **Formatting**: Use double-line spacing, British English spelling, and single quotation marks.
* **Data Policy**: Accepted papers require the submission of datasets and programs for replication.
* **Ethics**: A research ethics statement must be included in the Methods section if human subjects were involved.

## Features
- Pre-configured `authblk` for multiple authors/affiliations.
- Reference style set to author-year (e.g., Swinnen, 1997).
- Tables configured to avoid vertical lines as required.
